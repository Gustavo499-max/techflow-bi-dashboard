# TechFlow BI Dashboard

## Sobre o Projeto

Plataforma analítica desenvolvida para monitoramento de métricas de engenharia de software utilizando métricas DORA, modelagem dimensional e pipeline ELT.

O objetivo do projeto é permitir análise estratégica sobre produtividade, estabilidade operacional e impacto da IA generativa no fluxo de desenvolvimento.

---

## Funcionalidades

- Dashboard Executivo
- Métricas DORA
- Pipeline ELT
- Modelagem Dimensional
- Dashboard CTO
- Análise Estratégica
- Visualização de indicadores operacionais

---

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- Chart.js

---

## Estrutura Analítica

A solução foi construída utilizando arquitetura baseada em:

- GitHub API
- Jira API
- Data Warehouse
- Pipeline ELT
- Star Schema
- Indicadores DORA

---

## Objetivo Acadêmico

Este projeto foi desenvolvido como atividade prática de Business Intelligence aplicado à Engenharia de Software, com foco em observabilidade de engenharia e análise estratégica orientada a dados.
