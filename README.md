# TechFlow BI Dashboard

## Como rodar o projeto

### Opção 1 — Abrir diretamente
Basta abrir o arquivo index.html no navegador.

---

### Opção 2 — Rodar servidor local (recomendado)

Instale a extensão Live Server no VSCode.

Depois:

1. Abra a pasta no VSCode
2. Clique com botão direito no index.html
3. Clique em "Open with Live Server"

O dashboard abrirá automaticamente.

---

### Opção 3 — Rodar com Python

No terminal:

python -m http.server 5500

Depois abra:

http://localhost:5500

---
